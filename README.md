# 📊 Simulador de Impacto Socioeconómico - La Paz

> **SYS_STATUS: ONLINE // CORE_V2.0.26**
> 
> Motor de simulación interactiva para calcular y visualizar el impacto en la cadena de suministros, la canasta familiar y la disponibilidad de carburantes.

## 🎯 Objetivo del Sistema

Permitir la representación, cálculo y visualización del impacto real de los problemas logísticos, la especulación y los bloqueos en el territorio nacional mediante modelos matemáticos. Diseñado para facilitar la toma de decisiones con un enfoque estrictamente analítico.

## ⚙️ Características Técnicas

Desarrollo frontend estructurado bajo los requisitos técnicos requeridos:

*   **Estructura Semántica (HTML5):** Arquitectura validada utilizando `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` y `<footer>`.
*   **Manipulación Dinámica del DOM:** Renderizado de resultados en tiempo real mediante `innerHTML` y `textContent`.
*   **Gestión de Eventos:** Interacción controlada por escuchadores de eventos asíncronos (`onclick`, `addEventListener`).
*   **Motor Lógico y Alertas:** Evaluación matemática para determinar estados críticos de inventario y presupuesto.
*   **UI/UX Dinámica (CSS3):** Cambio de clases en tiempo real e interfaz de alto contraste inspirada en terminales de datos para optimizar la lectura de información.
