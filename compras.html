Módulo: Presupuesto de Compras Familiares

>> Contexto del Problema

Frente a la incertidumbre económica, las familias deben ajustar sus presupuestos limitados para adquirir alimentos básicos. Este módulo automatiza la verificación de solvencia, calculando si los fondos disponibles cubren la lista de compras planificada o si se requiere endeudamiento.

>> Conclusiones del Modelo

El déficit presupuestario mensual obliga a las familias a tomar decisiones críticas: reducir la cantidad de alimentos, buscar alternativas de menor calidad o recurrir a préstamos. Este modelo evidencia la fragilidad financiera ante precios inestables.
