# 📊 Simulador de Impacto Socioeconómico - La Paz

> **ANALYTICS_CORE: OPERATIONAL // DATA_FEED_2026**
> 
> Proyecto académico desarrollado para visualizar y calcular el impacto de las fluctuaciones en la cadena de suministros y la canasta familiar.

## 🎯 Objetivo del Sistema

El objetivo principal de este simulador interactivo es permitir a los ciudadanos representar, calcular y visualizar el impacto real de los problemas logísticos, especulación y bloqueos en el territorio nacional mediante modelos matemáticos sencillos. Busca facilitar la comprensión de la crisis y apoyar la toma de decisiones familiares de manera objetiva.

## ⚙️ Características Técnicas (Rúbrica Cumplida)

Este proyecto fue construido aplicando las mejores prácticas de desarrollo web moderno:

*   **Estructura Semántica (HTML5):** Uso riguroso de etiquetas como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` y `<footer>` para una arquitectura accesible y estandarizada.
*   **Manipulación Dinámica del DOM:** Inyección de resultados y cálculos en tiempo real utilizando `innerHTML` y `textContent`.
*   **Gestión de Eventos:** Integración de escuchadores de eventos (`onclick`, `addEventListener`) para una interactividad fluida con el usuario.
*   **Sistema de Alertas Lógicas:** Evaluación matemática de variables para determinar e indicar si el estado de inventario o presupuesto es **Normal** o **Crítico**.
*   **Estilos Dinámicos (CSS3):** Cambio de clases CSS en tiempo real (colores, alertas) dependiendo de los resultados matemáticos. Diseño de interfaz (UI) enfocado en alta legibilidad y contraste.
